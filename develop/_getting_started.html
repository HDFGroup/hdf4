Learning HDF4

There are several resources for learning about HDF4. The HDF Group provides an on-line HDF4 tutorial, documentation, examples, and videos. There are also tutorials provided by other organizations that are very useful for learning about HDF4.

The HDF Group Resources

For a quick introduction to HDF4 see the following:

IntroHDF4 A very brief introduction to HDF4 and the HDF4 programming model and APIs
LearnHDFView A tutorial for learning how to use HDFView. NO programming involved!
LearnBasics Step by step instructions for learning HDF4 that include programming examples
